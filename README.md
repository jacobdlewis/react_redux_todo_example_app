# lifeway-spa 

LifeWay SPA React Boilerplate

## Installation

Download node at [nodejs.org](http://nodejs.org) and install it, if you haven't already.

```sh
npm install lifeway-spa --save
```


## Tests

```sh
npm install
npm test
```
